#### What is Dart?
- Dart (https://dart.dev/)  is a modern, object-oriented programming language developed by Google. It is designed for building web, server, and mobile applications.
#### Below is list of the key features of Dart:
- Strongly Typed: Dart is a statically typed language, which helps catch errors at compile time.
- Asynchronous Programming: Dart has built-in support for asynchronous programming.
- Cross-Platform Development: Dart is the language used by the Flutter framework, which allows for cross-platform development of mobile, web, and desktop applications from a single codebase.
- Rich standard library: Dart comes with a rich standard library that provides a wide range of functionalities, including collections, file I/O, and more.
